# good-deeds-client

Backend for this app `https://github.com/AlexanderSUS/good-deeds-server`

### Install and run

1. Clone this repo
```
git clone git@github.com:AlexanderSUS/good-deeds-client.git
```

2. Go to repos folder
```
cd good-deed-client
```

3. Install dependecies with 
```
npm install
```

4. Run app with command
```
npm run dev
```
